Selectors that could be hard to understand


            element>element {}
            element+element {}
            element1~element2 {}

[attribute~=value] {}

One of separated by spaces value for example:

                yes for: [attr="value novalue"]
                no for: [attr="novalue"]

[attribute|=value] {}

Attribute starts with value

                yes for: "value" or "value-", or "value-2"
                no for: [attr="value123"]

[attribute^=value] {}

Attibute starts with value

                yes for: "value123", "value-123", "value"
                no for: "novalue", "no value"

[attribute$=value] {}

Attibute ends with value

                yes for: "novalue", "value", "no-value"
                no for: "value123"

[attribute*=value] {}

Attribute have value at any part

                yes for: "value", "novalueno", "value-123"

Box? Flex? Flexbox? what's all what you need


Use it for changing position any containers

That is really good and big theme

Flex for main container


  • flex-direction: row, column, row-reverse, column-reverse
  • flex-wrap: nowrap, wrap, wrap-reverse
  • flex-flow: {direction + wrap}
  • justify-content: flex-start, flex-end, center, space-between, space-around, space-evenly
  • align-content: stretch, flex-start, flex-end, center, space-between, space-around, space-evenly
  • align-items: stretch, flex-start, flex-end, center, baseline

Flex for self


  • flex-grow: 1 1 0 1
  • flex-shrink: 1 1 0 1
  • flex-basis: 50% 50% 50px 100px
  • flex: {grow + shrink + basis}
  • align-self: stretch, flex-start, flex-end, center, baseline
  • order: 2

What SCSS can do for us?

Variables



            $font-stack:    Helvetica, sans-serif;
            $primary-color: #333;

            body {
              font: 100% $font-stack;
              color: $primary-color;
            }

Nesting

Include a selector into a selector

            nav {
              ul {
                margin: 0;
                padding: 0;
              }
            } // nav ul

            a {
                &:hover {
                    color: red;
                }
            } // a:hover

Including

Made one file from two file

            @import 'reset'; // include _reset.css
            body {
              font: 100% Helvetica, sans-serif;
              background-color: #efefef;
            }

Inheritance

Get properties from one class and share

            %message-shared {
              padding: 10px;
            }

            .message {
              @extend %message-shared;
              border: 1px solid;
            }

            .success {
              @extend .message;
              border-color: green;
            }

Mixins

Group of repeating properties

            @mixin cool-border($color, $width: 1in) {
              border: {
                color: $color;
                width: $width;
                style: dashed;
              }
            }
            p { @include cool-border(blue); }
            h1 { @include cool-border(blue, 2in); }
            @mixin - =cool-border{}
            @include - +cool-border{}

Media

Use @media-queries where you whant

            .main {
                width: 300px;
                @media screen {
                  .sidebar {
                    @media (orientation: landscape) {
                      width: 500px;
                    }
                  }
                }
            }

Operators

We can use math operation +, -, /, *

            aside {
              float: right;
              width: (300px / 960px) * 100%; // 31.25%;
              color: #010203 * 2; // #020406;
              background: rgba(255, 0, 0, 0.75) + rgba(0, 255, 0, 0.75);
            }
            body {
                $font-size: 12px;
                $line-height: 30px;
                font: #{$font-size}/#{$line-height};
            }

Statements if, else, else if, for, each, while


            $i: 6;
            @while $i > 0 {
              .item-#{$i} { width: 2em * $i; }
              $i: $i - 2;
            }
            @each $animal in puma, sea-slug, egret, salamander {
              .#{$animal}-icon {
                background-image: url('/images/#{$animal}.png');
              }
            }
